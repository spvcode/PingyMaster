## 🔧 How To Use PingyMaster Commands

Here are some basic commands you can try with the bot:

- `/ping example.com`  
  Checks if example.com is reachable and shows response time.

- `/tcpcheck example.com 80`  
  Checks if TCP port 80 (usually for websites) is open.

- `/ip 8.8.8.8`  
  Shows details about the IP address 8.8.8.8.

- `/dns example.com`  
  Runs a DNS query and returns info like nameservers.

You just type these commands in the chat with the bot and wait for the reply.
